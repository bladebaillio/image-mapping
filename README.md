# Image Mapping

An extension on MakeCode Arcade for image mapping and simple shape-based transforms.
Use it to map textures into quads, map textures into color regions, and draw outlines.
The Topic for this extension can be found here: [https://forum.makecode.com/t/image-mapping-extension/]

## Block Groups
- Quad Mapping
- Region Mapping
- Outlines
- Query

## Quad Mapping

Use this group to project a source image onto a 4-corner shape.

<a id="map-to-quad"></a>
### map to quad

Draws the source image directly onto the target image using quad points.

<a id="mapped-copy-to-quad"></a>
### mapped copy to quad

Same mapping behavior, but returns a new image instead of changing the original target.



## Region Mapping

Use this group to map a source image into the shape of a selected color region.

<a id="map-to-color-region"></a>
### map to color region

Maps the source image into pixels of the target that match the selected color.

<a id="mapped-copy-to-color-region"></a>
### mapped copy to color region

Returns a new image with the color-region mapping applied.

<a id="region-size"></a>
### region size

Returns the width or height of the selected color region.


<a id="color-region-image"></a>
### color region image

Returns a new image that contains only the selected region color.

<a id="tile-map-to-color-region"></a>
### crop map to color region

Maps a repeating/cropped version of the source into the selected region with scale and offset controls.

<a id="cropped-copy-to-color-region"></a>
### cropped copy to color region

Copy-return version of crop mapping. The input target stays unchanged.



## Outlines

Use this group to draw outlines around all visible pixels or around one selected region color.

<a id="draw-total-outline"></a>
### draw total outline

Draws an outer outline around all non-background pixels.

<a id="outlined-transparency-copy"></a>
### outlined transparency copy

Copy-return version of total outline drawing.

<a id="draw-region-outline"></a>
### draw region outline

Draws an outline around pixels matching `regionColor`.

<a id="outlined-region-copy"></a>
### outlined region copy

Copy-return version of region outline drawing.



## Query

Use this group to build and edit quad point arrays.

<a id="set-quad-corner"></a>
### set quad corner

Updates one corner (`TL`, `TR`, `BL`, or `BR`) in existing x/y point arrays.

<a id="make-quad-x-points"></a>
### make quad x points

Creates the x-coordinate array for quad mapping.

<a id="make-quad-y-points"></a>
### make quad y points

Creates the y-coordinate array for quad mapping.
